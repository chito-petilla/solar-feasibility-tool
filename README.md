# Solar Feasibility Tool
A web-based Solar Return on Investment (ROI) calculator that estimates the financial feasibility of installing a solar energy system based on energy consumption, electricity cost, system cost, and efficiency.

This project simulates a lightweight renewable energy consulting tool used for early-stage feasibility analysis and investment decision-making.

## Overview
This system provides a simplified financial model to evaluate solar energy investments. It calculates expected savings and estimates the payback period based on user-defined inputs.
It is designed as a decision-support tool for energy analysis and sustainability planning scenarios.

## Functional Features
&nbsp;•	Monthly electricity consumption input  
&nbsp;•	Electricity rate-based cost estimation  
&nbsp;•	Solar system cost analysis  
&nbsp;•	Efficiency-based energy offset modeling  
&nbsp;•	Annual savings calculation  
&nbsp;•	ROI/payback period estimation 

## Calculation Logic
The system uses a simplified financial model: 
&nbsp;•	Monthly Savings = Energy Usage × Electricity Rate × Efficiency   
&nbsp;•	Annual Savings = Monthly Savings × 12   
&nbsp;•	ROI (Payback Period) = System Cost ÷ Annual Savings   
This provides a high-level feasibility estimate for solar investment decisions.
